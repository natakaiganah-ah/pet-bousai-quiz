# ペット防災チェックアプリ - Claude Code引き継ぎメモ

## これは何か
大分県版ペット防災チラシの内容をもとにした、診断クイズ形式のWebアプリ。
飼い主が23問の防災チェックに回答すると、優先度別配点で100点満点のスコアと改善アドバイスが出る。

## 現状
- Viteベースの React アプリ（`npm create vite -- --template react`）
- `src/App.jsx` に全ロジック・全23問のデータが入っている（単一ファイル構成）
- ローカルでの動作確認・調整はClaude（claude.ai）上のArtifactsで完了済み
- **まだGitHub Pagesへの公開は未実施**（このタスクをClaude Codeに引き継ぎたい）

## やりたいこと（ネクストアクション）
1. GitHub Pagesにデプロイして、公開URLを取得する
2. かおりさんのLINE公式アカウントで、そのURLを患者さん向けに共有する

## デプロイ手順の想定
```bash
npm install
npm run build
```
`vite.config.js` に GitHub Pages 用の `base` 設定（リポジトリ名のサブパス）を追加する必要あり。
例：リポジトリ名が `pet-bousai-quiz` なら

```js
export default defineConfig({
  plugins: [react()],
  base: '/pet-bousai-quiz/',
})
```

その後、`gh-pages` パッケージか GitHub Actions で `dist/` を `gh-pages` ブランチにpush。

## 設問データの構成（src/App.jsx 内 questions 配列）
各設問は以下のフィールドを持つ：
- `id`：設問番号
- `category`：カテゴリ名（健康管理・所有者明示・しつけ・慣らし等）
- `priority`：S（最重要）/ A（重要）/ B（標準）/ C（あると良い）
- `points`：配点（S=8, A=6, B=4, C=2）
- `petType`（任意）：`"dog"` or `"cat"` — 犬猫共通でない設問にスキップボタンを出すためのフラグ
- `headline`：「やってない」「わからない」回答時に表示する一言メッセージ（設問ごとに個別設定済み）
- `question`：設問文
- `praise`：「やってる」回答時の褒めメッセージ
- `advice`：「やってない」「わからない」回答時の詳細アドバイス

## 注意点・経緯
- マイクロチップに関する設問は依頼により削除済み（id=3が欠番）
- 優先度バナーは中央揃え・幅を狭めたピル型デザインに調整済み（依頼対応）
- 「一つ前に戻る」「最初に戻る」ボタンを各設問下部に実装済み
- スコア計算はスキップした設問を分子・分母双方から除外する仕様
- id=9（避難場所の確認）は、地域にペット同行避難所が少ないことを踏まえ、
  「避難所・親族や知人の家も含む」という文言に変更済み
